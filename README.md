# weatherApp
### An app for displaying current weather information of the location entered by the user.
## Tech Used
* Node.js
* Express.js
* HTML5
* CSS3
## Dependencies
* express
* handlebars
* postman-request
## APIs 
* Weather Stack
* Mapbox
* Fetch
